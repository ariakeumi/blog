OS X 10.10 Yosemite 黑苹果安装手记

2015年01月06日
阅读时长: 4 分钟

朋友为了玩FF14升级PC平台，然后把旧平台送我了，好朋友啊！！！

他的新平台：

CPU i7-4790K

GPU GTX970

MB MSI Z97

RAM Kingston 8GB DDR3

SSD Toshiba 256GB

Power Antec Neo Eco 550M

送我的旧平台：

CPU i5-2380P

GPU GTX550Ti

MB GA B75

RAM Corsair 4GB DDR3

得到新平台打算做的第一件事就是装上黑苹果，然后再玩GTA。以前的是Llano APU集显平台，装不了黑苹果，也玩不了GTA，虽然GTX550Ti也不一定能玩…

黑苹果对硬件要求很严格，技嘉的主板比较容易搞，但是我的是ITX机箱，还得再买了块技嘉1155接口的ITX板子。

1155的ITX板停产应该很久了，淘宝很少货，找到3个合适的。

二手GA-H67N-USB3-B3，价格325。

GA-B75N-WIFI，价格599。

GA-H77N-WIFI，价格599。

B75N-WIFI和H77N-WIFI主要是多了双网卡、无线网卡、蓝牙这些对我没用的功能，而且也不好驱动，大概…所以选了最便宜的二手H67。

ITX机箱装起来就是麻烦，线怎么理还是乱，把琴里扔进去。

在收到硬件前看了很多远景上黑苹果的帖子，先整理下比较重要的吧。

安装前：

黑苹果新手引导, 常见疑难解答, 必要知识普及帖

安装镜像：

Yosemite 10.10 和 10.10.1 正式版 懒人版安装镜像 支持变色龙 Clover引导 支持虚拟机

懒人版安装简单，合适我。

引导方法：

Windows下使用变色龙r2378+ 引导 OS X 10.10 懒人版安装超详细完整图文通用教程

【变色龙】支持 10.10 的 Chameleon_2.3svn_r2529 Mac版+Win版+EFI_Tools

Clover+EFI 好像很难，最终选择了变色龙+MRB引导安装， 使用EasyBCM添加wowpc.iso来安装变色龙引导，我用的是修改版的 Chameleon_2.2svn_r2402_10.10_CBv2_pkg+wowpc 。

详细安装方法：

在 win7 / win8 下安装苹果系统 (懒人版)

排错方法：

新手常见(五国)(-v图)错误解决(原版,破解kernel,补丁kext下载)

分区，写入安装映像、安装引导这些很简单，教程写的也很清楚，也不会出什么错，就不放图了。

安装的时候才是麻烦。

第一次安装，遇到Pci Configuration Begin错误，安装上面帖子的排错方法Q18。启动时加入『-v npci=0×2000 或 npci=0×3000』依旧不行，在org.chameleon.Boot.plist 加入

Kernel Flags
npci=0×2000

也不行，替换文件试了还是不行。

各种方法试了无数次还是不行，整个人都快崩溃了，最后确定放弃安装Yosemite，去安装OS X Mavericks试试，然后安装很顺利很快就搞定了。

显卡原生驱动，声卡没驱动，然后搞着搞着又进不了系统了，一怒下继续装Yosemite！

接着又是遇到那个错误，重复着搜索『Pci Configuration Begin』等关键词，终于爬到一张英文帖子。

其中一条回复，把10.9.5的的AppleACPIPlataform.kext替换进去，应该是这个意思吧。

我试着做，搜到10.9.5的AppleACPIPlataform.kext，替换进去，结果让人激动，跳过了PCI错误。

接着，新的错误又出现了。

NVDRstartup:Official 后黑屏了，只能强制关机。

进入Windows找到S\L\E下的NVDRstartup.kext删掉（备份好，安装完成还要替换回去才能驱动显卡），继续尝试进安装界面。

结果奇迹般的成功了，很顺利的安装了，安装完成后需要进入Windows复制文件，但是这时候是进不了Windows的，需要进PE用DiskGenius激活Win分区，我激活系统分区还是进不了，激活300M左右的那个系统保留分区就能进了。

把安装分区Extra目录复制到OS X系统分区目录下，然后重启 -v -f 进入设置向导界面。

设置完成系统就安装好了，接下来搞驱动。

用Multibeast，参照自己的硬件勾选然后安装重启就可以了。

声卡、网卡、USB3.0、SSD TRIM都成功搞定。

只有显卡驱动失败，3M显存，鼠标移动都卡出翔的感觉。

编辑变色龙的引导文件，开启显卡侦测，成功识别显卡。

但是分辨率、双显示器什么的都弄不了，还是一样卡顿。

然后我用Kext Wizard修改权限，重建缓存，执行然后重启。

最后，终于解脱成功了，硬件基本都驱动起来了。

以为这样完了，但是最后的最后我确定了，我这张技嘉的 H67N-UBS3-B3 竟然会啸叫，而且声音还不小，深夜的时候根本无法静下心来！

关掉了CPU节能选项，声音终于没了，但是睿频就没了，不甘心。

过两天退了这块主板入那块GA-H77N-WIFI吧。

累死了…休息。
