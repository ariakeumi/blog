Ubuntu 代替 Windows 的可能性探究

2014年09月02日
阅读时长: 4 分钟

Ubuntu是一个自由、开源、基于Debian的Linux发行版，发行周期为6个月，由 Canonical 公司和自由软件社区开发。 

不玩游戏、不使用专业大型软件、非程序员用户，试下看能不能用Ubuntu系统代替Windows。

# EFI下Win 8.1+Ubuntu 14.04安装教程

参照上面的教程EFI + GPT安装 Windows 8.1 / Ubuntu 14.04 双系统。 只需要一个U盘，很简单，引导是这样的。

Ubuntu的桌面，左边是启动器，状态栏在顶部，这种布局在16:9的显示器下看着很舒服，因为感觉视野大了很多，然后我把Windows的任务栏也固定在左侧了。

壁纸换成了星空图（自带的），很有Mac OS X的感觉。

可以用和Win类似的Win键 + 数字键打开任务栏（启动器）的程序。

版本号14.04 LTS，14.04代表是2014年04月发布，LTS表示长期支持，发布后18个月内的支持。

设备名 Abyss Gundam，深渊高达 ZGMF-X31S /RGX-02，出自机动战士高达SEED DESTINY，我的智能设备都用高达里面的机体命名，略中二。

这个叫什么来着…可以搜索本地和网络的内容，甚至Wikipedia，按Win键（Super L）呼出。

Ubuntu的软件中心，里面的东西少的可怜，排版也不好。

其实我在想，要是Windows中也有软件中心该多好（Metro那个渣渣商店不算啊），一个MS帐号同步所有安装的软件。 文件管理器，可以显示flac音频里面的专辑封面哦。 不知为何SSD的两个盘和HDD的盘图标不一样。 而且可以连接Android手机，我以为不可以来着。

Ubuntu自带的音频播放器，可以播放Flac格式的音乐，软件本身不能设置快捷键，需要到系统设置那里设置。

在linux下有个叫 Wine 的神器，可以让Ubuntu运行Windows的程序。 成功运行Foobar2000，但是全局快捷键用不了了。 Evernote也能正常运行，终于可以不用Web版的了。

自带Firefox浏览器，Chrome有linux版的，如果没有Chrome我会不知所措的。Ubuntu下的Chrome字体渲染有些奇怪，看起来不太舒服，自动更新到了64位的Chrome 37。

Ubuntu自带的视频播放器，播放1080P的MKV的BDRip，拖动有些慢，应付一般影音需求还是没问题的。

Ubuntu还集成了LibreOffice办公套件，非专业人士应付下应该没问题，现在我更多的用Google Docs作表格了，同步起来方便。 Ubuntu自带的中文输入法很蛋疼，明明是拼音输入法设置界面确实英文的，而且很多字或词打不出来，比如“qiong”（穹）字。

好在搜狗拼音输入法有linux版，顺便说一下，在Ubuntu 14.04 LTS版本中只需要双击下载的deb包就能直接安装，在Ubuntu 12.04中安装比较繁琐，甚至我下载linux版搜狗输入法后自动弹出帮助页面介绍安装方法，不过我是没法体验这么繁琐的安装方法了，貌似安装后需要重启才能用，而且没有词库同步功能。

搜狗输入法 For linux 安装指南

不过有国产厂商能发布linux的软件我已经很欣慰了。ORZ

腾讯曾开发过QQ For linux，那还是09年的时候，现在已经安装不了了。

导致Ubuntu现在没有原生的QQ客户端，只能使用Web QQ。

Web QQ没有自定义消息提示音，不能添加自定义表情，甚至不能发送图片，使用后战斗力下降998。

费了九牛二虎之力，终于找到了个能用的Wine QQ。

Linux兼容内核论坛 – wine qq 2013 for linux Ubuntu 64位兼容

可以用全局快捷键，可以截图，可以发送图片（虽然有点麻烦），资源占用有些高。

而且经常抽，比如打不开主面板，所以我才截聊天窗口的图。

一星期用下来，感觉发行版的linux使用门坎没传说中那么高。

至少在Ubuntu中，很多快捷键甚至和Windows的一样，设置界面也比Windows的控制面板易用多了。如果再有原生QQ客户端和更多的软件，当成主力系统也不是什么问题。
