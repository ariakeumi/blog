炒土豆丝の软件（Android）

2014年02月26日
阅读时长: 2 分钟

# Google Play & 酷市场

Play Store 无论设计还是使用体验都是最好的应用市场，可惜被墙，翻墙的话下载速度又比较慢，另外吐槽 App Store 真是难用死了，

酷市场则是国内最良心的应用市场，没事刷刷酷市场能发现很多新奇好玩的软件。

如果一款软件另有捐赠版/付费版，酷市场下载不了，但是评论里面都会有热心酷友给出捐赠版/付费版的下载链接，所谓同性交友网站…

# Nova Launcher

原生风格的启动器，功能强大不失小巧。

# PowerAMP

Android上音质最好的音频播放器之一，音效也是一流强悍。

这是我第一个舍得花钱买的软件，价格3.99刀，主要是破解比较麻烦…

我使用了一个叫 Poweramp skin KitKat/JB/ICS 的皮肤，有 Kitkat 的导航栏通知栏渐变阴影，播放界面呈现专辑封面的高斯模糊特效，非常漂亮。

# X-Plore File Manager

貌似 Android 最流行的文件管理器叫E.S管理器，好吧，我觉得E.S一点都不好用。

用过 Symbian 系统都该知道大名鼎鼎的 X-Plore 吧，它竟然还有 Android 版。

Android采用了双窗格树形界面，操作效率高。

可以连接 Google Dirve、Dropbox 和 OneDrive 等国外知名网盘，还有 SSH、局域网、FTP 和 Picasa 相册访问等功能。

# QQ 轻聊版

原版是屎，轻聊版没那么臭。

# EverNote

印象笔记之所以强大是因为它几乎全平台霸制，而且无论是桌面端还是移动端都很好用。

界面一看起来就让人觉得非常舒服，搜索功能依旧强大。

# 快图浏览

公认的 Android 最好用的本地图片浏览器。

# Google 相册

Google 相册则是刚从 Google+ 分离出的网络相册，无限储存空间。

# fqrouter

2014年中由于 Google IP 被大量封锁，导致翻墙速度极慢。

现在只能添加私人的 Shadowsocks 账号使用了。

# 什么值得买

购物优惠网站色魔张大妈（SMZDM）的 Android 客户端，遵循 Material Design 规范。

# Imagine Free

图片分享网站 instagram 的第三方客户端，原生客户端带底部 Tab 实在不能忍。

# CCleaner

Windows 上大名鼎鼎的清理工具。

# SeriesGuide

追剧助手，美剧、英剧、日剧应有尽有，再也不怕忘记看到第几季第几集了。

# AdAway

一款基于 Hosts 文件的 Android 广告屏蔽软件，同时 hosts 文件列表由开源组织进行维护，时刻保持最新，类似浏览器的 Adblock 插件。

之前介绍过一款 Xposed 下的去广告软件，叫 MinMinGuard ，通过API呼叫进行广告拦截，理论上拦截效果会更好。

# 水表助手

霸气的名字，其实是查快递的！

# 云颜文字

卖萌用。

# Xposed Framwork

神器当之无愧，Xposed Framwork 是Android系统一款可以在不修改APK的情况下影响程序运行（修改系统）的框架服务，基于它可以制作出许多功能强大的模块，且在功能不冲突的情况下同时运作。

Xposed 下的东西太多了，所以我单独写一个帖子介绍这货。
