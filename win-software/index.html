软件

炒土豆丝の软件（Windows）

2013年12月12日
阅读时长: 5 分钟

常用软件索引，免费软件一般放官网下载页面的链接，付费软件如果有破解版的话会放网盘链接。

  • # 装机必备

 

# Wox

可以安装插件的快速启动工具，功能更强大。

Google Chorme

三大浏览器之一，扩展程序功能接近FF，速度干掉IE，还有强大的同步功能+跨平台，除了比较占内存和字体渲染不够好基本没有其他缺点。

炒土豆丝のChrome扩展程序

PotPlayer

KMP作者出品，小巧，解码能力强大，绿色无广告。

Foobar2000

音质一流的音频播放器，小巧，简洁，可添加插件，功能强大。

作者Peter Pawlowski曾是Winamp的开发者，后来不满Winamp越来越往图形、外观发展而离开公司并开发了这款软件。历经了8年时间才脱离beta发布正式版。

作者没有制作官方中文版以及出汉化包，此版本由Asion Spa汉化。

EverNote

中文版叫印象笔记，个人经常用到的软件。免费版一个月有50MB流量，不同步很多图片的话应该够用。

全平台霸制，Windows，Android，IOS，Mac OS X，BB，甚至Web OS。还有个叫印象笔记·剪藏的浏览器扩展可以帮把喜欢的网页剪切同步到EverNote。

Google Picasa

谷歌的图片查看管理软件，配合XnView使用。

 

# 迅雷 9 去侧边栏版

迅雷极速版被砍，加速被河蟹。原版迅雷9变成带下载功能的浏览器，意见使用修改版。

Bandizip

支持图片预览的压缩软件，无广告界面清新，比WinRAR和7Zip好用。

CCleaner

小巧功能强大的清理软件，可清理各种程序缓存，注册表，还可卸载程序，插除驱动器。

Everything

神器，超级强大的硬盘搜索程序，Windows自带搜索实在太渣了。

另外这货在WIN8/8.1中必须设置管理员身份运行才能搜到东西。

Windows Defender

集成于Windows8/8.1中的微软自家杀毒软件 Microsoft Security Essentials。

  • # 硬件检测

Aida64

一款非常全面的硬件检测软件，还有温度监控，负载测试等，非常强大。绿色单文件破解版。

CPU-Z

著名的检测CPU的软件，几乎每一个硬件评测都能看到。

GPU-Z

CPU-Z的兄弟。

Fritz Chess Benchmark

国际象棋，计算CPU一秒能下多少步象棋，一款很古老的跑分软件，小撸怡情。

CineBench R11.5

跑分软件，比国际象棋专业很多，也臃肿很多。

3DMark 11

著名的跑分软件，主要测试显卡，收费，可以免费试用部分功能，还有跑分名人堂。

MemTest

内存错误检测，用于超频和蓝屏排查。

DisplayX

检测显示器色彩，坏点的小软件。

HD Tune Pro

著名的硬盘检测软件。

# CrystalDiskInfo

查看硬盘健康状态，一般用于SSD。

AS SSD Benchmark

很常见的SSD跑分软件。

DiskGenius

功能强大的硬盘管理工具，有分区，分区表备份恢复，数据恢复，MBR GPT分区转换等功能。

  • # 大型软件

Adobe系列软件绿色汉化版

有PS，PR，AE等众所周知的软件。

Microsoft Office Professional Plus 2013 

微软的办公套件，这个就不用说了，使用HEU KMS Activator激活180天，可重复。

另外说下，Windows 7/Vista系统可以使用Windows Loader激活。（我指的是系统）

VMware Workstation 10

VM的虚拟机软件，功能强大，绿色原生中文。

使用里面的批处理安装或者卸载服务，还要导入里面的注册表激活。

  • # 多媒体

# Easy Paint Tool SAI

SAI，日本的一款绘图软件，主要配合数位板来作画。

# GifCam & Ulead GIF Animator 5 & XnView

GIF制作，压缩和查看三件套，详细介绍。

Riot

图片压缩软件，支持批量压缩。

Bandicam

视频录制软件，可录制多种格式编码的视频，资源占用小，录制的视频体制也小，我的视频都是用它录的。

安装后右键以管理员身份运行注册机即可注册。

Mediacoder

通用视频音频格式转换工具，设置比较复杂不适合新人使用。

小丸子工具箱

可以抽取，压制视频，我一般用它把视频封装成FLV。

VideoCacheView

可以提取浏览器的缓存视频等文件。

MKVToolNix

MTK封装软件，也可以用来分离音频，添加字幕。

格式工厂

国人开发的转码软件，效率低下但使用简单，适合新手。

  • 美化

Rainmeter

雨滴，著名的美化软件，可以修改出更好非常漂亮瞎眼的主题。

不过我是简洁党所以并没有使用它。（桌面只有回收站的图标）

OblyTile

修改WIN8中的开始菜单磁贴图片的小软件。

Windows 7 Boot Updater

可以修WIN7/Vista的启动画面，一直没有找到修改WIN8/8.1启动画面的软件，唉。。。

  • 其他

 

硕鼠下载器

以前使用的iDown失效，改用硕鼠。

绿色版，运行绿化后，去官网输入视频链接就可以下载视频。

MangaMeeya

本地漫画阅读器，用看图软件看漫画是非常不明智的选择。

Sublime Text 3

非常漂亮的文本编辑器，被称为码农神器。不会编程的我只能哪来代替记事本，比WIN自带的记事本好用很多。

单窗口多标签，还有超级漂亮的界面。

还有热退出功能，什么是热退出？就是当你修改了文件，然后退出Sublime Text，它仍会保存那段你已经修改了的文本，即使断电也不怕改好的文本消失。

不过这货收费的，还贼贵（70刀），不过不注册也可以使用，不过保存文件时偶尔会提示你要注册。

已有汉化和破解  ，觉得麻烦的话用NotePad++代替也不错。

Easy File Locker

文件隐藏工具，总有不想被别人看到的东西。另外这只是个隐藏工具，没有加密功能，容易被破解。英文版，使用超级简单，可惜不支持GPT磁盘。

NT6 Hdd Installer

非常简单好用的系统安装工具。

Dism管理器

清理WIN8/8.1更新后的残留文件。

ShadowsocksR

第三方 SS 客户端，翻墙工具，自备服务器。

UltraISO

制作U盘启动的，还可以修改镜像。由卡饭版主ikimi编译的单文件注册版。

Recuva

小巧文件恢复软件，和CCleaner为同一公司出品。

Aegisub

跨平台开源的免费字幕编辑软件。许多字幕组皆采用这套字幕编辑软件，替视频制作非官方或非商业的字幕。

Hash 2.0 

MD5+SHA1文件效验工具。

FileLocatorPro

全文搜索，偶尔会用到，很强大。

NetSpeedMonitor x86  & NetSpeedMonitor x64

在任务栏显示流量监控的小程序，WIN8需要右键“兼容性疑难解答” – “测试程序” 然后才正常安装。

Tournament-Table

一个比赛对阵视图软甲，由我个人汉化。

SIS网址获取程序

啊咧，好像混进了奇怪的东西。。。

轻轻一点

轻轻一点找里番。

EverAver Renamer & EMDB

AV媒体库。

  • 过气

Launchy

快速启动工具，Windows的任务栏可以放常用软件，靠 Win键 + 数字键启动，对于那些不常用的，用Launchy比用开始菜单或者切到桌面打开快捷方式更简便，只需要记住软件的名字的前几个字符就可以了。

迅雷极速版

比原版小巧很多。

GoAgent

利用GAE服务器当代理的翻墙软件，速度快而且稳定，居家旅行必备。

Clover

把Windows的资源管理器变成Chrome的样式，增加类似Chrome的多标签页和快捷键功能。
