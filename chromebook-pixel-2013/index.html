ChromeBook Pixel 2013 折腾

2015年08月19日
阅读时长: 4 分钟

ChromeBook Pixel 和 MacBook Pro 是我最喜欢的笔记本，两者都是铝合金一体成色的外壳，正面和背面无开孔，超薄机身，完美的工业设计，同时它们价格都很吓人。2013 年版的 ChromeBook Pixel 低配价格高达 1299 美元，MBP 也差不多是这个价格，曾经考虑过二手的 MBP ，Retina 屏的基本都要 5K ，实在太贵了！

Pixel 则从未想过要买，以全新价格对比的话肯定会买 rMBP 的，性能和功能都更强。

然而前几天在闲鱼闲逛的时候突然发现一台价格不错的 ChromeBook Pixel ，虽然成色较差，正面一处划痕一处凹痕，背面多处小划痕，无箱说，但是看在才两千的价格上没多想就买了。

一年多前，我曾经花了一千六百多的价格买了一台 Acer 的 ChromeBook C720 ，但是那台本子还没拆封就卖了，因为觉得实在没什么亮点。这台 ChromeBook Pixel 就不同了，它有 3：2 比例的 2560 x 1700 超高分辨率触摸屏，极其适合网页浏览，光这个就值了。

  • # 外观

侧面面一个充电接口，一个 Mini DP 接口，两个 USB 接口和一个音频接口。

另一侧只有一个 SD 卡接口，我刚好有一张。

因为不是 Windows ，自然没有 Win 键。这样的话安装其他 Linux 发行版使用上也有些不便。

  • # Chrome OS

Chrome OS是由Google所进行的一项轻型电脑操作系统发展计划，发展出专用于互联网的云操作系统。该操作系统设计计划于2009年7月7日发布，系统植基于谷歌浏览器及Linux内核。最初设置在上网本上使用，在2011年上半年正式发表上市“Chrome OS”可以在x86或ARM两种微处理器上运行。
Google声明至2009年底，“Chrome OS”将以“Chromium OS”之名采用开放源代码。虽然“Chrome OS”植基于Linux内核，不过它会使用“一种新的系统”而不会采行目前正在使用中的Linux标准视窗系统，比如X窗口系统。

Chrome OS 的桌面就是一张 Android Material Design 风格的壁纸和任务栏，按 Alt 键和数字键 1 可以启动 Chrome 浏览器，跟其他桌面系统的 Chrome 一样，就是设置页面多了些 Chrome OS 的设置项。

Chrome OS 支持的格式：

  • Microsoft Office 文件：.doc、.docx、.xls、.xlsx、.ppt（只读）、.pptx（只读）。
  • 视频：.3gp、.avi、.mov、.mp4、.m4v、.mkv
  • 音频：.mp3、.m4a、.ogv、.ogm、.ogg、.oga、.webm、.wav
  • 图片：.bmp、.gif、.jpg、.jpeg、.png、.webp
  • 压缩文件：.zip、.rar
  • 其他：.txt、.pdf（只读）

原生音频播放器可以播放 FLAC 无损格式的音频，但是功能简陋，专辑封面不能显示，而且只能通过打开音频文件来打开播放器。

Chrome OS 是基于 Linux 的系统，进入开发者模式就可以用 Linux 的 Shell 了，按 Ctrl + Alt + T 。

启用开发者模式: Acer C720 Chromebook (简体中文) – ArchWiki

要用 Chrome 自然离不开翻墙，Chrome OS 有 Shadowsocks 客户端。

第一次登录的时候是要连接 Google 帐户的，这时候可以用过 Android 的翻墙路由器共享网络给 ChromeBook ，或者用带有 Shadowsocks 的路由器固件的网络。

安装 Shadowsocks 客户端比较麻烦，还要进入开发者模式，在 Shell 下操作，安装各种依赖。

Chromebook/ChromeOS安装Shadowsocks扩展教程 | Dogfight360

Linux 上没有原生 QQ 和 Evernote ，两者都有 Web 端，Evernote 的 Web 端很好用，但是我笔记很多，各种乱七八糟的，用 Web 端还是很不便的。前段时间 Evernote 被官方移植到了 Chrome OS 上，在 Chrome 应用商店的离线应用里面可以下载到。

说是移植，其实就是直接用 Android Runtime 运行，用 ARC Welder （APP  Runtime for Chrome）可以把  APK 格式的 Android 应用程序安装文件转换为 Chrome 程序，在 Windows 、 OS X 、Linux 等平台的 Chrome 浏览器中都可以运行，也就是说可以自己“移植” Android 应用，虽然能正常运行的不多。

ARC Welder 谷歌官方安卓模拟器 – 在电脑 Chrome 浏览器上运行 Android APK 软件游戏

官方版的 QQ 运行不了，第三方使用 Web QQ 协议的可以，比如 MaterialQQ Lite 和 MingQQ ，不过既然能用 Web QQ ，为何要用使用了 Web QQ 协议的 Android 应用？反正在 Chrome OS 是不能愉快的使用 QQ 的…

早就听说 ChromeBook 的触摸板很好用，是真的。

  • # 总结

低电压的 i5-3427U 发热巨大，功耗不好，加上超高分辨率的屏幕，导致续航只有 5 小时，键盘手感不佳，Chrome OS 软件缺乏，不能作主力使用。要是能安装黑苹果或者 Windows 就完美了！
