Windows 10 技术预览版 体验

2014年10月02日
阅读时长: 3 分钟

10月1日，微软开发布会宣布新一代操作系统Windows 10，而不是盛传已久的Windows, Windows TH, Windows X, Windows One，甚至是Windows9。在开发阶段曾泄露出新一代Windows内部代号为Windows Threshold。

微软旨在服务客户，称Windows 10为“有史以来最全面的操作系统（most comprehensive platform ever）”。新系统恢复了类似Windows 7的传统开始按钮，但是在细节上糅合了Windows 8和Windows 8.1的微软“Modern瓷砖”设计。Windows 10作为微软下一代系统的统一品牌名称，将覆盖所有尺寸和品类的Windows设备，可无缝运行于微型计算机（类似英特尔伽利略）、手机（ARM芯片）、平板（ARM和x86芯片）、二合一设备、桌面电脑及服务器等几乎所有硬件。Windows 10贯彻了“移动为先，云为先”的设计思路，三屏一云，多个平台共用一个应用商店，应用统一更新和购买，是跨平台最广的操作系统。新系统的名称跳过了一个数字“9”，标志着它实现了一个飞跃，旨在统一计算和移动设备体验。微软表示，今天发布的Windows 10是一个早期版本，微软仍旧在致力于优化平板电脑上的操作体验。Windows 10是一款为家庭、为企业、为个人量身打造的通用平台操作系统。Windows 10将会在在2015年晚些时候面向用户发售。

等了一天多终于来了，Windows 10 技术预览版开放下载，64位简体中文的镜像达到了3.9GB。

还是老样子，把ISO镜像写入U盘安装。

家里的EFI + GPT的机子安装失败了，提示安装源有问题，F**K！

在MBR的机子安装成功，安装起来和Windows8/8.1一模一样。

安装完成后进入桌面，那个Welcome to Tech Previ..的图标只是一个网页链接。

之前关于Windows 10的爆料已经很多了，虚拟桌面、开始菜单、Metro应用窗口化什么的都知道了。

但是安装完成后还是让我很兴奋。

最明显的一点，应用程序的边框像智能手机一样超薄了。

下图很明显看出左右两侧和底部变得很窄了，这个简直比Aero Glass更惊喜。

预览版什么的是不用激活的，连安装序列号都不用，2015年4月16日过期，那时候估计RTM就出了。

资源管理器中多了个主页，显示最近浏览文件、最常使用文件夹和收藏夹，使用快捷键Win + E默认打开资源管理器的主页，非常不适应！！！

最人性化 的多任务切换视图（Task View），当任务多时会变成多排显示，单个视图大小也是随动的。

在Windows Vista/7/8/8.1中的多任务切换视图预览图非常小，对近视者来说非常反人类，甚至不如Windows XP中的纯图标视图来得方便。

传闻中的虚拟桌面（多重桌面），不怎么感冒。

新的开始菜单，对我来说就是用来搜索。

窗口化的Metro应用 – 邮件。

我所使用的软件全部兼容Windows 10，传说中的通知中心和Cortana并没有出现，再过一段时间等新的预览版再安装比较好。
