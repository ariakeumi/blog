卡硬工具箱魔改教程

2018年04月08日
阅读时长: 2 分钟

卡硬工具箱是一款硬件检测软件工具，前身是卡巴工具箱，是显卡吧基佬们开发的出来硬件检测工具，提供BIOS工具、CPU检测、超频工具、性能测试、内存工具等众多常用工具。

这个工具箱已经一年多没更新了，里面的软件有些过时了，比如 CPU-Z 检测不出 Ryzen 处理器。

如果要升级里面的软件，很简单，把新版本的软件替换到卡硬工具箱的 program 目录下对应的软件储存目录上就行了，exe 文件和原来的命名一样就可以。

因为很喜欢这个软件，所以我研究了下怎么把给卡硬工具箱添加新软件。

修改卡硬工具箱需要用到 DB Browser for SQLite 。

下载它安装打开并点击 “打开数据库” ，选择卡硬工具箱目录下的 data 目录下的 set.db 文件。

浏览数据表，可以看到 catlst 表里就是分类菜单信息：

  • id 分类 ID，主界面的分类也是根据这个排序
  • catname 分类名称
  • icopath 分类图标，在 ui 目录的对应皮肤目录下
  • bgcolor 分类的背景颜色，十六进制颜色码
  • stamp 帮助文档链接，无视
  • blk 主界面分类菜单磁贴，1 表示小磁贴，2 是宽磁贴

需要添加分类菜单磁贴就仿照上面的格式新建记录。

选择 softlst 表新建记录添加自己所需的软件。

  • id 用于排序
  • catid 对应 catlst 表的分类菜单id
  • softname 软件名称
  • icopath 软件图标路径
  • softpath 软件路径
  • description 软件介绍

软件图标可以用 软媒魔方 的软媒图标大师提取，修改完点击 “写入更改” 就可以看到成果了。

这就是我修改完的卡硬工具箱，添加了很多自己需要的东西，放在U盘随身带。
