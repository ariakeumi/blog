海美迪 H7三代 电视盒子调教

2015年09月18日
阅读时长: 3 分钟

小米盒子用着越来越不爽了，播放 1080P 或者高码率 720P 带 DTS 音轨的电影时音频经常卡顿，暂停跳进度条也很卡，双核 A9 也该淘汰了。

最不能忍的是启动时竟然有广告，偶尔还推送广告，连屏保都是广告！ROOT 了禁用自带的启动器用第三方启动器，还不能设置壁纸！WTF！

反正是不会再考虑小米的东西，前段时间和朋友买了个海美迪的 H7 二代，感觉还不错，遥控器还能切换鼠标模式。

现在这家伙出了三代，处理器升级到了瑞芯微的64位八核 RK3368 ，当然，是那种大小核心的架构，八个核心不会同时开启，不过四个 A57 核心同时开启应该还是很给力的。

颜色挺土鳖的，带一条天线看起来有点像路由器，不过并不支持 802.11ac （5GHz）WIFI ，当然我都是用有线连接的，速度稳定。

还带着莲花口有点无力吐槽。

有线网口是百兆的，蓝光原盘的码率大概 50Mbps ，够用。

海美迪 H7 的遥控器密密麻麻的很多按键，小米的简洁很多，另外一个是功放机一个是电视机的遥控。

这遥控是我最满意的地方，它有一个带学习功能电视机控制区，意味着它可以同时开电视机和功放机，淘汰另外两个很大却没什么用的电视机和功放机遥控器，电视机遥控器值用来开电视机，功放机偶尔调节音量，四个学习按键刚好，没有功放机的可以用它来遥控空调！

底下的数字区可以用来输入密码和设置快捷切换应用，也很好用。

另外，小米的遥控器确认键不能 “暂停 / 播放” MX Player，这点有点奇怪，海美迪的可以，满分。

最讨厌的默认启动器（桌面），没啥用，而且不能设置第三方启动器为默认启动器，需要 ROOT 后用钛备份冻结掉它默认启动器才能更改。

自带的媒体中心，不能自动匹配字幕有点可惜，其他功能都有。

Android 5.1.1 ，很原生，Lollipop 的彩蛋都有。

5.1 声道音响系统的需要设置 HDMI 音频源码输出，由功放机解码。

用了第三方启动器（Nova Launcher）就不能打开海美迪的媒体中心了，因为找不到快捷方式。

但是第三方启动器比自带的好看多了，能设置壁纸，添加自己喜欢的小部件，遥控器操作也没问题。

通过 DS Video 和 MX Player 来播放和管理 NAS 的电影/剧集，可以通过射手的 API 匹配字幕。

以前的 XBMC 多媒体中心，功能超级强大，但是我这里输出 5.1 声道音频的时候超级卡不知为何，放置 Play 。

可通过 TV盒子工具 进行 ROOT ，先用 Adb工具 开启 Adb ，然后通过 TCP 连接到电视盒进行一键 ROOT 操作，傻瓜式的。之后就慢慢折腾吧。

4K 啊什么的就不测试了，毕竟电视都是 1080P 的，4K 的片源也找不到，反正盒子性能很强大，应用切换播放暂停拉进度条什么的都反应快速。

比较硬伤的是，海美迪的关机真的是关机，所以每次打开盒子都要开机，设置成休眠会死啊…另外关机还是开机的指示灯都闪瞎狗眼…

瑕不掩瑜，性能无可挑剔，很好用电视盒子。
