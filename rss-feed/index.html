# 何为RSS？

RSS（简易信息聚合）是一种消息来源格式规范，用以聚合经常发布更新数据的网站，例如博客文章、新闻、音频或视频的网摘。

简而言之，就是把订阅的网站信息聚合在一起方便阅读的工具。有了RSS，不必再去苦苦寻找，只要打开RSS阅读器信息自己就会到来！

# 什么值得买

购物优惠信息聚合网站，每天50+以上的信息，适合重度购物狂，可以省（花）很多钱。 特别是每日白菜系列，经常手贱，千万不要订阅！！！_(:3ゝ∠)_ SMZDM的海淘和物品选购攻略也超级赞。

BBC中文网

BBC的新闻比较客观，看国际新闻的话很不错，一般新闻都很简短。

月光博客

知名的IT博客，我建博客就是受月光影响，文章质量都很不错。

小众软件

软件推荐博客，我最喜爱的软件这专题必看（软件票选排行）。

和邪社

ACG向新闻站，经常有AV女优访谈并赠送光盘。

原站不提供全文的RSS输出，使用下面的链接可以订阅全文RSS。

http://fullrss.net/a/http/feed.feedsky.com/hexieshe

# 编程随想

罕见的政治博客，墙外网站，访问需梯子。作者是码农，思维极其缜密，分析能力极强。

原本应该是个IT博客，后来开始写GFW相关文章，现在比较多国内政治分析文章，写的有条有理，材料都来自官媒，比轮子什么的强多了。

强烈建议订阅，RSS的好处就是墙外的无需翻墙也可以看。

福利控

顾名思义，提供各种福利的网站，戒撸别订。

電腦玩物

主要介绍Evernote、Firefox和Google等软件使用经验的博客，作者是TW人。

善用佳软

同电脑玩物，更新很慢，文章超详细，进阶级。

Engadget 中国版

AOL旗下的科技博客，比较关注移动互联网。

因为一条具有助纣为虐性质的文章使我退订。

# cnBeta

代替Engadget，国内的IT新闻站，文章比较多，一天可能有近两百篇IT新闻，适合重度IT控。

如何订阅？

使用Feedly订阅，登录Feedly后，右上角的搜索。

找到订阅源后点“+Feedly”就订阅了，虽然界面全英文都是上手很容易。

手机是非常适合看RSS的，建议使用Press阅读，体验绝佳。
