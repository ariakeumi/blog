【PairLens Vision】ACG壁纸 及 萌音主题

2013年01月29日
阅读时长: 2 分钟

原帖来自CHH作者为烧耳机的lolicon！
因为CHH的图片不能外链弄了好久的说！
其实这主题的萌化程度很低不过系统声音好棒！

Windows 7 Starter 初级版 / Home Basic 家庭基础版 用户无法安装任何第三方主题;

请一定 先安装系统声音, 后安装系统主题, 否则全套有声主题的良苦用心可能体会不到了~

( 并且还包含 鼠标指针, ICL: icolibrary 等. 硬盘图标为 可选安装项, 另紫色文件夹并不包含其中. )

PS:( 楼主实力有限,请勿与 动漫东东 上那些第三方主题制作高手进行比较 )
刚才有人问我这些主题除了壁纸还有什么不同。。其实。。声音都不同用这主题为的就是声音！

主题预览

«Anohana»

«Chakai»

«Chobits»

«Fate» & «Saber»

«Vocaloid»

«Nymphet»

使用方法：

解压声音文件到 C:WindowsSubject 就行了。
然后双击安装主题就大功告成了，其实这套主题Win8也可以使用。
在C盘 Windows 文件夹里新建 Subject 文件夹，把主题声音的文件夹拖进去就可以了。（好像不能用ZIP压进去）
还有Win8用起来没有登录声音注销声音等。

最后，Win8娘登场！

此主题并非官方原版，原版需要将区域时区改为日本！
此版本双击即可安装，并且Win7用户也可以使用。
声音不错，微软良心。
